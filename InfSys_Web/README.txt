Installation :
1) Veuillez créer le dossier "testPShell" dans "C:\".
2) Installez Node.js (https://nodejs.org/en) à cet emplacement "C:\Program Files\nodejs"
3) Installez Power Automate (Microsoft Store) puis créez un flow nommé "InfSys"
4) Copiez le contenu du fichier à cet emplacement "[Répertoire téléchargement]\InfSys\InfSys_Web\Power_Automate\Main.txt"
5) Clique droit sur l'onglet "Main" dans Power Automate puis coller
6) Créez un subflow dans Power Automate et nommez le "Subflow_Web_Page"
7) Copiez le contenu du fichier à cet emplacement "[Répertoire téléchargement]\InfSys\InfSys_Web\Power_Automate\Subflow_Web_Page.txt"
8) Clique droit sur l'onglet "Subflow_Web_Page" dans Power Automate puis coller
9) Copiez le dossier situé ici "[Répertoire téléchargement]\InfSys\InfSys_Web\InfSysPowerAutomate" à cet emplacement "C:\testPshell\InfSysPowerAutomate"
10) Lancez le programme
